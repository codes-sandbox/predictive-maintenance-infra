# 予兆検知インフラ基盤 (Predictive Maintenance Infra)

## 概要
センサーデータを安全に蓄積し予兆検知を行うためのクラウドインフラをTerraformでコード化（IaC）した
手作業による環境構築を排除し構成管理の自動化を目的としている

## システム構成
- **ストレージ**: AWS S3 (センサーデータ保存用)
- **管理手法**: Terraformによる宣言的なインフラ管理

## 設計のこだわり
- **IaCの推進**: インフラをコードとして管理することで環境の再現性と構成の可視化を担保した
- **機密保護の意識**: `.gitignore` を適切に設定しTerraformのステートファイル等の機密情報がリポジトリに含まれないよう配慮している
- **拡張性を考慮した設計**: 現状はシンプルな構成ですが機能単位でモジュール化が可能な構造を意識しており今後のリソース増加にも柔軟に対応できる設計を目指す

## 今後の展望
- インフラ構成のモジュール化による再利用性の向上
- S3データを処理する推論基盤の自動デプロイ機能の追加
