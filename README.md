# 予兆検知インフラ基盤 (Predictive Maintenance Infra)

## 概要
センサーデータを安全に蓄積し予兆検知を行うためのクラウドインフラをTerraformでコード化（IaC）した

## システム構成
- **ストレージ**: AWS S3 (センサーデータ保存用)
- **管理手法**: Terraformによるモジュール化設計

## 設計のこだわり
- **モジュール化**: `modules/` 配下に機能単位でコードを切り出し、再利用性と拡張性を高めている
- **再現性**: 手動構築を排除し誰でもコマンド一つで環境を立ち上げられるようにする

## 今後の展望
- 将来的にはS3のデータを処理する推論基盤の自動デプロイ機能を追加予定
